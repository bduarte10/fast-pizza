## Stack utilizada

**Front-end:** React, Redux, TailwindCSS

## Instalação

Clone este repositório para o seu ambiente local

Navegue até o diretório do projeto

Execute:

```bash
  npm install 
```
    
## Rodando localmente

Inicie o servidor

```bash
  npm run dev
```
